## 📌 Overview

The **Employee Management System (EMS)** is a scalable web application designed to simplify employee administration and workflow tracking within an organization.

It provides **role-based dashboards** for:

- 👨‍💼 **Admins** — Full system control  
- 👨‍🔧 **Employees** — Task and profile management  

---

## ✨ Key Features

### 🔐 Authentication & Security
- JWT-based authentication  
- Role-based authorization (Admin / Employee)  
- Secure password hashing  

### 👨‍💼 Admin Functionalities
- ➕ Add new employees  
- ✏️ Update employee details  
- ❌ Delete employees  
- 📋 View all employees  
- 📝 Assign and manage tasks  
- 📊 Monitor performance  

### 👨‍🔧 Employee Functionalities
- 📌 View assigned tasks  
- 🔄 Update task status  
- 👤 Manage profile  
- 📈 Track personal progress  

### 🌐 General
- 📱 Responsive UI  
- ⚡ Fast REST API  
- 🛡️ Error handling & validation
